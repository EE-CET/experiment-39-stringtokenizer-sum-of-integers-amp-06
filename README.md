# Experiment 39: StringTokenizer (Sum of Integers)

## Problem Statement

Write a program that reads a line of integers and then displays each integer and the sum of all integers.
**Requirement:** Use the `StringTokenizer` class to parse the input line.

## Input Format

* A single line containing integers separated by spaces.

## Output Format

* Print the sum of the integers.

### Example 1

**Input:**

```text
10 20 30
```

**Output:**

```text
60
```
